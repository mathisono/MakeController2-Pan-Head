00216   The values to use to create a mask to pass into any of the \b Bits style functions.  
00217   Your mask values need to be of type longlong since it needs to represent 64 bits, 
00218   for the 64 IO lines.

00221

00223   longlong mymask = 0;
00224   mymask |= (IO_PA00_BIT | IO_PA03_BIT | IO_PA11_BIT);
00225   // now disable the pullups for lines PA00, PA03, and PA11
